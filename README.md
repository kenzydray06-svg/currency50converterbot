# Currency50 Converter Bot

A Telegram bot for currency conversion.

## Commands

- `/start` - Welcome message
- `/help` - Help guide
- `/convert 100 USD EUR` - Convert currency
- `/rates` - Show exchange rates
- `/list` - List all currencies

## Deploy

1. Get token from @BotFather
2. Set TELEGRAM_TOKEN environment variable
3. Deploy on Railway

## Environment Variables

- `TELEGRAM_TOKEN` - Your bot token (required)
